
# Invoice Management PRO

## New Features
- Bootstrap UI
- Role-based admin (super admin / staff)
- Invoice PDF (dompdf)
- Export Excel (CSV)
- Multi-currency
- REST API
- WhatsApp auto reminder (cron)

## Requirements
- PHP 8+
- MySQL
- Composer (for dompdf)

## Default Admin
admin / admin123 (SUPER_ADMIN)
